KatOS

A graphical OS built for privacy.

Released: Not for a while (I WILL make this happen someday, even if years into the future, I want to)



KatOS would be a graphical OS built for privacy, but also functionality.

It'd be a great alternative to Windows and Linux, as I would ensure it can run Windows and Linux apps as well as our own custom file type.

Unlike Windows, it would not require a user account.

I actually have attempted this already and have gotten a kernel and bootloader to work (not quite graphical though), but it just isn't feasible by myself yet.

I also tried to compile the Linux kernel for it, but it whined at me.

So one day, I'll just do it myself (or with a team, who knows) and make it the best alternative you can choose.
