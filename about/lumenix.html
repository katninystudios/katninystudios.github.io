Lumenix

A custom browser engine built around web standards.

Released: Not for a while, if ever



Lumenix would be a independent web engine built on W3C standards.

I did attempt this and I even have my progress on GitHub, but all it can really do is render HTML. It's a good HTML viewer, not so much a good web browser.

If I ever do it (or just fork an engine and start working on it independently from there, I'm not Ladybird, I didn't say I wouldn't), it will power the Luma browser.
